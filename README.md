# 🏍️ Velocity Moto

A high-speed 3D motorcycle racing game built with Three.js. Dodge traffic, collect fuel and nitro, complete missions, and unlock new bikes across multiple tracks.

## Features

- **4 Unique Bikes** — Starter, Cruiser, Sport, Superbike with distinct stats and visual models
- **Multiple Tracks** — City highway, desert, arctic, and neon environments
- **Cockpit Dashboard** — Analog speedometer, fuel/nitro gauges, gear indicator
- **Mission System** — Tiered mission chains with manual redeem rewards
- **Daily Rewards** — Login calendar, streak milestones, and daily task challenges
- **Garage** — Custom paint colors, upgrade system, stat visualization
- **Traffic & Hazards** — Dynamic traffic spawning, environmental creatures, road obstacles
- **Responsive Controls** — Keyboard (WASD/Arrows) + mobile touch support

## Controls

| Action | Keyboard | Touch |
|--------|----------|-------|
| Steer | A/D or ←/→ | Swipe left/right |
| Accelerate | W or ↑ | Auto |
| Brake | Shift | Brake button |
| Nitro | Space | Nitro button |
| Camera Toggle | C | — |
